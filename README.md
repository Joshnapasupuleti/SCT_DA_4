# Marketing Campaign EDA

## Objective
Analyze a marketing campaign dataset and recommend which marketing channels should receive more budget based on ROI.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Key Findings
- Facebook achieved the highest average ROI.
- Website and Google Ads performed similarly.
- ROI differences across channels were relatively small.

## Recommendation
Allocate slightly more budget to Facebook while maintaining investment across other channels due to the small ROI differences.
